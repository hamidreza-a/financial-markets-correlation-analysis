# Financial Markets Correlation Analysis

This project analyzes monthly asset returns and correlations between major equities, Bitcoin, and inflation using Python.

## Overview
- Collected financial time-series data using the yfinance API
- Performed data cleaning and manipulation with pandas
- Computed monthly returns and correlation matrices
- Visualized time-series trends and asset correlations

## Tools
- Python (pandas, yfinance, matplotlib)
- Jupyter Notebook
